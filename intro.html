How to Play???


  • Cows and Bulls is an old code-breaking mind game which dates back to a century or more.

  • There will be an autogenerated random number of "4 digits" stored in and the user needs
    to guess using as less trials as possible.

  • Everytime the user enters a number,there will be a display of number of cows and bulls
    in the number the user has entered.

  • Cows denotes the number of digits that are present in the original number and Bulls
    denotes the number of digits which are in the exact position as in the original number.

  • For example the original number is 1953 and the number enetered by the user is 2936
    the number of cows is 2(which are 9 and 3 as are same as in the original number) and bulls is 1(which is 9 as in same place).

  • Enter a 4 digit number with no repetitions

Have fun!! Thank you
